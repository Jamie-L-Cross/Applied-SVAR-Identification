# Applied-SVAR-Identification
A Short Course on Applied SVAR Identification with Applications in MATLAB.

The course uses Ambrogio Cesa-Bianchi's VAR Toolbox.

To run the applications:
1. Download the 'VAR-Toolbox' Toolbox from 'ambropo' here on GitHub
2. Create a new folder called 'Applications'
3. Download the files from this repository and place them in the Applications folder
4. Unzip and run
